Connect your Google Sheet to sync lists between both phones. Only needs to be done once.

Step 1
Create a Google Sheet
Go to sheets.google.com, create a new blank sheet, and share it with Sina (Editor access).
Open Google Sheets →
Step 2
Open Apps Script
Inside your new sheet, click Extensions → Apps Script. A new tab opens with a code editor.
Step 3
Paste the script
Delete everything in the editor. Paste the code from the Code.gs file you downloaded. Click the 💾 save icon.
Step 4
Deploy it
Click Deploy → New deployment
• Type: Web app
• Execute as: Me
• Who has access: Anyone
Click Deploy, approve permissions, then copy the URL it gives you.
Step 5
Paste your URL below
The URL looks like: https://script.google.com/macros/s/ABC.../exec
